Our Project

Dr Holly Tootell’s research project aims to investigate the design process of the Jump Start application, which is a development tool on physical activity and gross motor skills for pre-schoolers and parents and educators. In order to achieve this, various tests have been conducted on paper, with results written down by hand.

Our team’s job is to make the testing and data collection process easier for the researchers.

Our proposed solution to this problem is an online web application that aims to make it convenient and efficient to conduct the tests. By using a web application we can have the tests conducetd on devices such as tablets and automatically collect data and compile results into useful graphs and tables. A web application also allows for easy deplyment to preschools and allows for instructions to be displayed through our application to inform educators on how to conducts tests.

Users

Researchers
Researchers refer Dr Tootell's research team. Their role in the system will be making the tests and analysing the results provided from the tests.
Pre-schoolers
The pre-schoolers are the major focus of the research. As such they will be interacting with the system through tests in order to collect their results and behaviours.
Educators
The Educator's main role is to setup and guide the pre-schooler's through the test. Educator's are an important factor due the preschooler's being familiar with them.

Issues

Data Collection
Currently the tests are conducted on paper with results written down by hand, this involves difficulty in data collecting as every result has to be recorded by hand and then transformed to be put into a database. This limits the effectiveness of how many tests can be taken and with data being physically taken it is possible for human error to occur.
Deployability
Currently as tests are paper the research team needs to make their way down to a preschool to deploy their tests as they need to explain to educators on how to conduct these tests and how to collect correct data. This overall limits the deployability of tests which means overall means less data for researchers to go off.
