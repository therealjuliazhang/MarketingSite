Usability Testing Platform For Preschoolers.



Main features

create

Create / Edit Test

Enables flexible data collection practices for researchers working with pre-schoolers in IT design.

people

Manage Users

You can easily add pre-school educators or research supporters so you can expand your research area.

insert_chart

Analyse Results

Evaluating digital products, feature or prototype with preschoolers. Test data can be exported as csv file to review.

devices

Conduct Test

Effective user testing platform for preschoolers to support the conducting of tests, recording of test data as well as the instruction of the adult supervisors.

How it works


  Administrator Side 

View all existing tests

Administrators can view all the existing tests they have created so far.
This includes details such as date created, date last modified and a detailed description for each test.



Create and edit tests

Administrators can create tests as well as edit existing tests.
While editing tests, administrators can preview tasks and remove them as necessary.


View Results

Administrators can review group or individual test results.
Test results can be sorted by several categories, including test title, location of test, and gender or age of test participants. Test results can also be exported as a csv file for later analysis.


Manage users

Administrators can decide who is allowed to sign up for an account.
New users are listed under Pending Users and will be unable to access the system until an administrator accepts (or declines) them.

  Educator Side

Likert Scale

The Likert Scale allows pre-schoolers to choose whether or not they like the monster being displayed. To make choices easier for the pre-schoolers, the likert scale has been simplified to provide two options only – a happy or a sad face


Identify The Body Part

Preschoolers can be asked to identify a body part of the monster. The points that the pre-schoolers touch on the screen will be saved as coordinates that can be displayed accurately later when viewing results.


Character Ranking

In Character Ranking, pre-schoolers can rank their favourite monsters. The system collects the choices made by the children and ensures that there is a clear favourite by testing characters with the same point score together.


Preferred Mechanics

Preferred Mechanics combines paper testing with digital data entry. Preschoolers are asked to interact with a piece of paper with a monster printed on it. To save time in data collection, educators are able to enter data directly into the system rather than writing results down by hand and entering it into a spreadsheet later.
