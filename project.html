Childs play

Usability Testing Platform For Preschoolers.



Main features

create

Create / Edit Test

Enables flexible data collection practices for researchers working with pre-schoolers in IT design.

people

Manage Users

You can easily add pre-school educators or research supporters so you can expand your research area.

insert_chart

Analyse Results

Evaluating digital products, feature or prototype with preschoolers. Test data can be exported as csv file to review.

devices

Conduct Test

Effective user testing platform for preschoolers to support the conducting of tests, recording of test data as well as the instruction of the adult supervisors.

How it works


  Administrator Side 

View all existing tests

Administrator can view all the existed test they have created so far.
The administrator can check the time each task they first created and the last modified，also can review the detail description for each task



Create and edit tests

Administrator can create the test and edit the current test
In the edit test administrator can preview each test and remove the test if they wants


View Results

Administrator can review the group or individul test result, export as csv file,also can sort the result by test title,location,gender,age,etcs.


Manage users

Administrator can accept the educator and administrator register request

  Educator Side

Likert Scale

In the likert scale test
the preschooler can chooose whether they like the monster by click the smile or sad face


Identify The Body Part

In the identify the body part test
Educator will give a instruction and the preschooler needs to follow the instruction click the right body part of the monster


Character Ranking

In character ranking test
Preschooler can rank their favorite monsters，this task will need educator know which monster is most favorite


Preferred Mechanics

For the preferred mechanics test
the educator can record the preschooler operations when they interact with the image
